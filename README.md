## Local Development

Simply open `index.html` in your browser, or use a local server:

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```

Then visit `http://localhost:8000`

## Deployment

Since this is a static site, you can deploy to:

### Vercel (Recommended)
```bash
npm i -g vercel
vercel
```

### Netlify
Drag and drop the folder to netlify.com/drop

### GitHub Pages
1. Push to a GitHub repository
2. Enable GitHub Pages in Settings
3. Select the main branch as source

### Custom Domain (whoswe.xyz)
Point your domain's DNS records to your hosting provider:
- For Vercel: Add the domain in project settings
- For Netlify: Add the domain in site settings
- For GitHub Pages: Add a CNAME file with your domain
